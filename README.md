# NU ITCS Program Tree Monitor 🌳

A modern, interactive web application built for Nile University (NU) students to visualize and track their academic progress through the ITCS (Information Technology and Computer Science) curriculum. 

**🔗 [View Live Demo Here](https://mostafasaber-25.github.io/nu-itcs-tree/)** 

## 🚀 Features
* **Interactive Curriculum Map:** Automatically generates a visual prerequisite tree for courses using Mermaid.js.
* **Progress Tracking:** Click on courses to mark them as completed. The app calculates your remaining GPA credits, total degree credits, and courses done.
* **Local Storage:** Your progress is automatically saved to your browser, so you can close the tab and return later without losing your data.
* **Multiple Majors Supported:** Includes tracks for Computer Science (CS), Artificial Intelligence (AI), Biomedical Informatics (BMD), and Cybersecurity (CSEC).
* **Modern UI:** Features a sleek, glassmorphism dark theme built with Tailwind CSS.

## 🛠️ Built With
* **HTML5 & CSS3**
* **JavaScript (Vanilla)**
* **Tailwind CSS** (for styling and responsive design)
* **Mermaid.js** (for rendering the prerequisite flowchart)

## 💡 How to Use
1. Select your specific ITCS major from the top cards.
2. Scroll down to view your specific program requirements and the interactive prerequisite tree.
3. Click on any course node in the tree to toggle its status (Completed / Not Completed).
4. Watch your progress statistics update in real-time!
5. Use the "Reset Progress" button if you ever need to start over.
